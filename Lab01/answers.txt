6.1
- What happens if users choose “Cancel”?
- If a user clicks the "Cancel" button, the second dialog will display: "You've chosen: No".

- How to customize the options to users, e.g. only two options: “Yes” and “No”, OR “I do”
and “I don’t”?
- To customize the options to users, instead of using "showConfirmDialog", we will use
"showOptionDialog", because showOptionDialog allows coder to customize button text
